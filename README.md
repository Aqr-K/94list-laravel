本程序是免费开源项目，核心代码均未加密，其要旨是为了方便文件分享与下载，重点是 GET 被没落的 PHP语法学习。
开源项目所涉及的接口均为官方开放接口，需使用正版 SVIP 会员账号进行代理提取高速链接，无破坏官方接口行为，本身不存违法。
